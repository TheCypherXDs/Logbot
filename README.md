# LogHunter Discord Bot

Продвинутый Discord-бот для логирования, аудита и модерации сервера.

## 🚀 Возможности

- Логирование сообщений (удаление, редактирование)
- Логирование голосовой активности
- Система предупреждений (/warn, /warnings, /delwarn)
- Модерация (/clean)
- Профили пользователей (/profile)

## ⚙️ Установка

```bash
pip install -r requirements.txt
python main.py
